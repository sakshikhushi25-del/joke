# 😂 Joke Generator Web App

## 📌 Project Overview

This is a simple **Joke Generator Web Application** built using JavaScript, HTML, and basic DOM manipulation. When the user clicks a button, the app fetches a random joke from an external API and displays it on the screen.

---

## ⚙️ How It Works

* A button is dynamically created using JavaScript.
* When the button is clicked, a request is sent to a joke API.
* The API returns a random joke in JSON format.
* The joke (setup + punchline) is displayed on the webpage.

---

## 🚀 Features

* Generate random jokes with one click
* Uses async/await for API handling
* Dynamic DOM element creation
* Simple and clean UI design

---

## 🛠️ Technologies Used

* HTML5
* JavaScript (ES6)
* Fetch API

## ▶️ How to Run

1. Download or clone the project folder
2. Open the `index.html` file in your browser
3. Click the **"Generate a joke"** button
4. A random joke will appear on the screen
